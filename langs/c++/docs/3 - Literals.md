## Literals/Constants

# Interger Literals
- Whole Numbers
- int decimal = 100     // Decimal
- int octal = 0123      // Octals starts with 0
- int hex = 0xfg4..     // Hexadecimal starts with 0x

# Floating-Point Literals
- Number with decimal points or exponential notations
- float num1 = 3.14     // Decimal
- double num2 = 2.5e3   // Exponential ( 2.5 * 10^3 )

# Character Literals
- Single character enclosed in single quotes
- char letter = 'A'

# Boolean Literals
- True or False
- bool isValid = true;

# Escape Sequence Literals
- Special characters represented using backslash
- printf("Hello\tWorld\n");
- \t = Tab
- \n = Newline
